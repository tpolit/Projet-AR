Description détaillée

Structure représentant un processus participant à l'élection du leader.

Paramètres
mpi_rank: rank mpi
state: etat du processus
initiateur: true = candidat, false sinon
vg: voisin de gauche
vd: voisin de droite
nb_in: nombre de tokens qui sont revenus
etape: numero de l'etape, utilisé pour calculer distance = 2^etape
chord_id: id chord du processus
